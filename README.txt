#registroEstoque

Sistema para registro de solicitaçoes de alteração de Ordens de Produção e Estoque

Este projeto tem como intuito de registrar em um banco as solicitaçoes diárias de alterações de OPs e Estoque, com intuito de registro e identificação de quais areas estão precisando de mais ajustes.
Com isso posso rastrear quais itens estão dando mais problemas na hora do fechamento, identificando problema na baixa e consumo das Ordens de Produção
Projeto feito para uso interno na empresa que trabalho
